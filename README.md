## **Introduce**
> ### Go beyond impossible
안녕하세요! **임베디드 개발자**를 목표로 공부중인 **김서영**입니다.

주로 **IoT**와 **로봇공학** 분야에 관심을 갖고 폭넓게 공부해 보는 중입니다.

아직 배우는 중에 있어 부족한 점이 많지만, 즐거움과 호기심을 원동력으로 꾸준히 나아가고 있습니다. 

여러분과 함께 활동하며 많은 것을 공유하고 성장할 수 있으면 좋겠습니다.
